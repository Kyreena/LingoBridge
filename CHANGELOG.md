# Changelog

All notable changes to LingoBridge will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2026-03-02

### Added
- **MAJOR UPDATE**: Expanded ASL library from 56 to 261 words (366% increase!)
- Added comprehensive pronoun vocabulary (I, me, we, us, he, she, they, it, this, that)
- Added 30+ common verbs (eat, drink, sit, stand, walk, run, come, go, see, look, hear, etc.)
- Added family and relationships (mother, father, sister, brother, family, friend, boy, girl, man, woman, baby, child, people)
- Added 11 colors (red, blue, green, yellow, orange, purple, pink, black, white, brown)
- Added food and drink vocabulary (water, milk, juice, coffee, tea, bread, fruits, snacks)
- Added educational terms (homework, test, quiz, exam, study, practice, question, problem)
- Added academic subjects (math, science, english, history, art, music)
- Added classroom objects (computer, paper, pencil, pen, desk, chair)
- Added 15+ emotions and states (angry, mad, excited, surprised, tired, sick, hurt, pain, afraid, scared, worry, sorry)
- Added common objects and places (house, home, car, bus, train, phone, door, window, table, bed, bathroom, kitchen, store, library, hospital)
- Added time vocabulary (day, night, week, month, year, time, hour, minute, morning, afternoon, evening)
- Added all days of the week (monday through sunday)
- Added numbers 11-20, hundred, and thousand
- Added 30+ more adjectives (new, old, young, hot, cold, warm, cool, fast, slow, easy, hard, difficult, right, wrong, correct, same, different, more, less, many, much, little, few, all, some, every, none, nice, great, fine, ok)
- Added useful connecting words (ready, wait, again, repeat, remember, forget, important, try, use, share, turn, change, keep, stay, leave, arrive, excuse)

### Changed
- Updated README badges to reflect 261 available signs
- Updated statistics display to show 261 available signs
- Updated help documentation with new word count
- Reorganized library structure with clear category comments

## [1.0.0] - 2026-02-19

### Added
- Real-time speech recognition using Web Speech API
- ASL video library with 56 common words and phrases
- Sequential video playback for recognized words
- Visual highlighting of words with ASL signs available
- Responsive UI design for desktop and tablet
- Comprehensive test suite
- Browser compatibility checks
- Detailed documentation in README.md
- Dark mode toggle for better accessibility
- Demo mode for testing without microphone
- Download transcript functionality
- Keyboard shortcuts for main actions
- Statistics display (words recognized, signs played)
- Settings panel with user preferences
- Professional metadata and SEO optimization
- MIT License
- Contribution guidelines
- Deployment instructions

### Categories
- **Greetings**: hello, hi, goodbye, bye
- **Courtesy**: please, thank you, yes, no
- **Classroom**: teacher, student, learn, book, read, write, class, school
- **Questions**: what, where, when, who, why, how
- **Actions**: help, understand, know, think, want, need, like
- **Time**: today, tomorrow, yesterday, now, later
- **Numbers**: 1-10
- **Adjectives**: good, bad, big, small, happy, sad
